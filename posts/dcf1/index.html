我在用的一些主力生产力、学习工具分享

本文最后更新于：15 分钟前

这篇文章的想法来自我的一位朋友问我平时在使用什么软件，简单交流之后决定写个文来介绍介绍。鸽了好久的博客终于开张了，希望看到文章的你能有一些收获。

开始构思这篇文章的时候总是想怎么分类，以什么顺序来介绍这么多软件和服务。绞尽脑汁，放弃了这种分类，决定用意识流来写，想起哪个写哪个，相信自然会有一些必要地联系。

TickTick

我最常使用的软件，第一个就是 TickTick，国内版名称滴答清单。我用来管理我平常的任务，日程，也用来捕捉日程生活中的奇思妙想，好心情，坏心情。现在我的生活几乎不能没有它。

全平台，不会掉链子。移动端配合 Siri 添加任务，方便！桌面端直接快捷键创建新任务，不会打乱办公的节奏，更不会错过每一个精彩的想法。

「再多提一嘴，我特意废弃了滴答清单所提供的「今天」清单，因为一旦有了「今天」这个概念，就反而会让人忽略「下一步行动」这个清单，误以为「今天」（隐喻着「今天的事情最紧急」）清单列表里的事情是今天最值得做的，一旦有了期限，就又有了压力。」
from 吕立青_JimmyLv

推荐阅读： 《搞定》最佳实践：如何利用滴答清单（TickTick）实践 GTD 理念？

Obsidian

知识管理工具 Obsidian，最近掀起一股风的 Roam reserach 的免费替代品，开源社区驱动，我很喜欢。现在在做我的知识体系构建软件。

双向链接关联起来每篇笔记的关键点，形成一张网，一眼就可以看到最重要的部分，不断保存笔记，关联笔记，逐步建立起个人的知识体系。使用存储在本地的 Markdown 文件作为存储介质。管理、迁移非常方便。

这里分享前几天看到的翠翠的博文，2020 年，或许你应当试试这样记笔记。很好的传达了我想说的话（我的水平写不出来，太失败了！）。

MoneyWiz 3

全力个人记账工具，功能非常强大，可生成 30 多种报表，深度分析消费情况。对账用的小对钩也是很贴心的功能，好好利用，基本不会漏掉一笔。

XDpv.png)

回想起来，记账已经快两年了，当初只是为了看看每一笔钱都花在了什么地方，坚持每天不差一分一厘的记着。到现在，随着大佬们的文章，记账体系变了几次，对于记账也有了新的理解，从只是一味的记录，到慢慢使用报表分析自己的消费习惯，再到利用预算功能规划自己的钱包，每一次进步都会带来不同的感受。为了学习使用 MoneyWiz 诸多强大的功能，还买了李大超人Leo 的从零开始做好个人记账。

我日常只在电脑上记，不需要付费版的同步功能，曾在 Setapp 上试用过一段时间，同步的效果也不尽人意，一端修改过多设置时就可能会发生同步冲突的问题。

一天结束，翻出来微信、支付宝的消费记录，一笔一笔记上去，还怪有成就感的。（不过，对于省钱、理财，还是没有什么进展😬。

Ulysses

优雅的文本写作工具，值得拥有。本文就是在 Ulysses 的帮助下完成，使用体验真的是很棒啊！！

非常喜欢在打字机模式 focus 到句子上，写的时候很专注于当前的一句，也很利于逐句的对文章进行修改校对。需要浏览上下文时，只要随便滑动页面，就会暂时取消打字机模式，很方便的。

只上架苹果全家桶，并使用 iCloud 同步。也可管理本地文件，写博客利器，增加你的产量。

优雅。

XMind ZEN

脑图工具，一直从老版本用到新发布的 ZEN 版，至今还在试用，真是太良心了，我只做知识整理、构建大纲用，输出时有水印与我无瓜，还能给软件宣传宣传，挺好。

要说他和其他脑图工具相比有什么优缺点，我也没有研究，只是个人一直在用。

Reeder

几个月前开始折腾 RSS 订阅，先后试玩部署过 Tiny Tiny RSS、Miniflux。听闻 Reeder 4 功能强大，界面漂亮，风格沉稳，遂直接入手。后来因 TTRSS 订阅不太稳定，服务器也不长时间持有，就没有重度依赖，转为 Reeder 本地账号订阅了。截止本文编辑时，未读订阅源还有 385 个，唉，曾经那个清空 Inbox 的少年回不来了。（阅读各路大佬文章看到的 RSS 订阅截图，还没见过未读为零的🤪）。

也是苹果生态内软件，只提供 iOS、iPadOS、macOS 版本。软件并不提供多端同步，如果需要可使用自搭建（TTRSS）或者 Feedly 等服务。（Reeder for macOS 免费领取了，需要多端同步的我权衡之下加入了 Feedly 的大家庭，本来想用 Inoreader 的，不过免费版有广告这不能忍吧）

就在写本文的过程中，Reeder 4 全平台限免啦，oh my god，试问谁能抵得住这股诱惑，冲啊，兄弟们。

RSSHub 是一个太棒的项目了，make RSS great again，吹爆 DIYgod。大神们适配了各种平台服务，高度整合了各种消息推送，可以节省好多时间。推荐给每个人。

Spotify

两个月前上车了 Spotify，真香！拼车的价格完全可以接受，我主要听欧美歌曲也基本不会出现歌不全的情况。多平台接力播放很舒服的，在电脑上写代码听歌，出门直接转到手机上播放，顺滑。

平时学习，写文的时候都会听歌，所以也就写进来了。

最近港区、日区都上线了歌词功能，不知道什么时候轮到美区，期待🥰。

Notion

也是最近半年了被吹爆了的效率工具，不过我并不是太喜欢，原因有三：

  1. 加载有点慢。实在是难以忍受几秒的加载时间。其实也没有多慢，但就是感觉很慢。
  2. 学习曲线有些陡，经常性劝退。
  3. 无法离线，倒不是离线有多重要，非要不可，而是这种使用体验要差一些。

所以我只在上面维护几个小项目，像我的书库啦，观影记录啦什么的，曾想在上面写周报，不过垃圾如我，失败了。

VS Code

巨硬出品，值得信赖。各种文本的编辑器，有时跑个测试代码。真是强大，开源社区驱动，配合各种插件，随手组装一个 All in one 工具出来。

IntelliJ IDEA

在学 Java，在这上面做一些 Java 项目。用起来感觉要比 Eclipse 舒服一点，虽然两个都挺慢的。所以一些简单的练手代码都用 VS Code 写了。

手机上用的一些小工具

除了上面列举的有移动端 App 的软件，我还在使用下面这些小工具。

Apple Podcasts

这半年赋闲在家，闲来无聊用播客填补生活的空洞。后来发现这也是一种非常有效的获取知识、前沿信息的途径，利用声音这种特殊的介质传播，更具亲和力，更加易于融入生活中。

随便说几个我平时听的博客吧：

  • 津津乐道：朱老板与各个领域的达人共同制作的博客节目。旗下好几个栏目，「科技乱炖」科技向、「津津有味」吃货向、「拼娃时代」教育向等。
  • 随机波动 StochasticVolatility：三个媒体人做的一档博客节目，有关社科、文学、并讨论一些社会热点问题。剩余价值炸号后复出博客
  • 故事 FM：用你的声音讲述你的故事。亲历者自述声音节目。我感觉这种形式非常不错，亲身经历的人来自我讲述，一些情感、故事的表达是无可比拟的。
  • 生活漫游指南：听爱科普的土豆和大家一起扯。
  • Anyway.FM 设计杂谈：新晋标签「赔钱博客」，两位设计师谈天说地聊设计。

Apple Books

原来用 Google Play Books，无奈中文无法换字体，并且只能使用翻页的阅读方式。买了 iPhone 后立即换到了 Apple Books 上，多种字体选择，翻页，滚动阅读都可以，黑暗模式也很优雅的说。爱了爱了，就靠好的工具来促进我的阅读量呢！
