给定一个字符串，找出其中不含有重复字符的 最长子串 的长度。

输入 输出 解释
示例 1 “abcabcbb” 3 因为无重复字符的最长子串是 “abc”，所以其长度为 3。
示例 2 “bbbbb” 1 因为无重复字符的最长子串是 “b”，所以其长度为 1。
示例 3 “pwwkew” 3 因为无重复字符的最长子串是 “wke”，所以其长度为 3。

少废话，先看东西😶

class Solution {
    public int lengthOfLongestSubstring(String s) {
        int start = 0, res = 0;
        HashMap<Character, Integer> map = new HashMap<>();
        for (int idx = 0; idx < s.length(); idx++) {
            if (map.containsKey(s.charAt(idx)) && start <= map.get(s.charAt(idx)) )
                start = map.get(s.charAt(idx)) + 1;
            // 更新字符上一次出现的下标
            map.put(s.charAt(idx), idx);
            res = Math.max(res, idx - start + 1);
        }
        return res;
    }
}

思路

要找出不含重复字符的最长子串，就要从重复字符下手，定义一个哈希表来存储每一个字符出现的位置，如果再次见到这个字符，就要更新他的位置到新的下标，并从新的位置来计算是否可以达成一个最长子串。

需要使用一个 start 变量来存储当前子串的开始下标，并且如果出现了重复字符，start 也要更新到新的下标。重新开始计算子串。

与此同时，要有个计数变量 res 保存子串的长度，但是并不需要跟着主循环递增，只需要计算 index - start + 1 就好，而且每次循环与 res 取较大值赋 res 即可。
