一、准备

下载各种包

  • RADIO-oscar-201803170847.zip ，oscar 底包
  • MK90.0-oscar-201904131310-NIGHTLY.zip ，魔趣每夜版全量包
  • open_gapps-arm64-9.0-micro-20190406.zip ，谷歌 micro 版全家桶
  • Magisk-v18.1.zip ，Magisk 18.1 (稳定版)
  • ssr-android.apk ，SSR 安装包，你懂的

注： 将以上全部包放入 U 盘或 SD 卡中，因为这次要完全重刷，手机内置存储也要清除

备份手机文件

  • 把轻课表的课表文件拉到了电脑上，同时注册了账户，进行了云备份
  • 把纯纯写作的备份文件拉到了电脑，同时注册了 Dropbox 基础版，云备份实现

二、开始重刷

  1. 关机，插入你的 U 盘或 SD 卡

  2. 按住 音量+ 键和 电源 键进入 TWRP

  3. 点进 Wipe => Advanced Wipe ，选中以下选项，五清

    • Dalvik / ART Cache
    • Cache
    • System
    • Data
    • Internal Storage

    注： 一般人 双清 就足够了，我这是特殊需求。

  4. 进 Install ，刷 底包 。刷完双清 ，不要重启

  5. 继续刷 全量包 ，刷完双清

  6. 然后刷入 Magisk 开机调试好 Magisk 再刷 open Gapps

    记： 刷入 Magisk 以后直接再刷 open-Gapps 会闪重进 Recovery 。实际好像是 Magisk 没有成功刷入。

  7. 重启体验

三、开机后的调试阶段

Magisk

调试失败，重回「一」。

原因：无法通过 ctsProfils 检测，后安装了 busybox & MagiskHide Props Config 依旧无效，尝试终端 root 运行 props 命令改 device fingerprint ，然后炸了，Lawnchair v2 一直停止运行，然后无奈，重刷之。

可安装 Magisk-v19 ，已正式发布。安装 MagiskManagerMD2.apk ，美化界面。

垃圾，升级不了，卸之，回原版。

第二次，ctsProfile 还是过不了，算了，不理那个了。

安装各大神器

  • Google Chrome ：怎么少得了呢。
  • Storage Redirect (存储重定向) ：可将一些软件在 /sdcard 下胡乱拉的屎（有些其实也不算屎，但是也在目录下产生了）重定向到相应包名的 data 下和 Android/data 下，保持 /sdcard 的清洁。
  • AppOps ：安卓权限管理工具，限制毒瘤应用随意获取权限。
    • Shizuku Manager ：。
  • Dir ：垃圾清理工具，可以以 占坑 的方式阻止一些应用不断生产垃圾。
  • SSR ：你懂的。
  • Telegram ：高雅的 IM 软件。
  • Google Translate ：装好以后可开启全局选中查单词，好用又不臃肿。
  • Share 微博客户端 ： 鸡仔 出品，我认为最好的微博客户端，没有之一。
  • 知乎 ：「学习」必备……。
  • Pure Writer (纯纯写作) ：我认为安卓端最好用的写作软件，是他让我有想在手机上写东西的冲动。（没有电脑的情况）
  • XiaoMi Push Framework (小米推送服务) ：可使一些应用使用小米系统级推送服务，节省电量。
  • FastHub ：一款 GitHub 安卓客户端。做得不错，解决了 GitHub 手机网页版的一些痛点。

其他小设置

  • 设置安全选项，屏幕锁定，指纹。
  • 开启系统样式随为 自动 (一天中的时间) ，这样晚上的时候系统会变成暗色，护眼又漂亮。
  • 开启谷歌自动换壁纸，一天一个样，每天好心情。

一些记录
