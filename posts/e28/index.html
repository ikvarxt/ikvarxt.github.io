9102 年，搭建博客的成本已经趋于 0 了，不需要任何钱钱，免费的服务到处都是，记录我的博客搭建过程。

Hexo 和 GitHub Pages 给我一切。感谢。

本地 Hexo 环境搭建

软件基础：macOS Mojave 。Hexo 中文文档 https://hexo.io/zh-cn/docs/

Hexo 基于 Node.js ，可利用 brew 命令安装 Node.js ，安装 npm 管理器。

brew install node
brew install npm

安装 hexo

npm install -g hexo-cil

创建一个文件夹来存放博客的文件夹 blog

mkdir blog

初始化 hexo 网站目录

hexo init blog
cd blog
npm install

创建第一篇博文

标题为引号内的内容。

hexo new "hello my blog"

这时你就可以编辑位于 ~/blog/source/_post 目录下的 hello-my-blog.md 文件。写写你的想法。

使用 hexo s ，浏览器访问 http://localhost:4000 本地预览你的博客。适合网站发布前进行调试。

我用 Typora 编辑博文， macOS 下一个十分好用的 Markdown 编辑器，亮点就是可以实时预览，还有众多主题可选。当然，你也可以直接用 Vim 来写。

部署至 GitHub Pages

  1. GitHub 中新建一个仓库，仓库名需为 username.github.io (username 为你的 GitHub 账号名)，才可在公网访问。GitHub Pages

  2. 安装 git deploy 插件。Hexo 官方文档

npm install hexo-deployer-git --save
  1. 修改博客本地根目录下的 _config.yml 配置文件。使 Git deploy 可用。
deploy:
    type: git
    repo: https://github.com/username/username.github.io

repo: 需改为你的 GitHub Pages 仓库地址。

  1. 终端执行 hexo clean && hexo d 即可将自己的博客上传至 Git 仓库。就可以在浏览器访问了。

    第一次上传无需执行 hexo clean ，不过以后都是需要先清空生成的文件才可以部署。

细节更改

主页信息

Hexo 配置 可以修改根目录下的 _config.yml 来加入你的特殊信息。网站名，简介，关键字，作者等主要信息都在这里存储。

博客主题

Hexo 主题 ，这里有很多优质的主题，可以挑选一个搭配自己的性格，如果有能力也可以自己写一个主题，上传到这里，让更多的人用。
