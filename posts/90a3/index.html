把辽宁工程技术大学课程表加入日程

本文最后更新于：1 分钟前

前段时间用 Scriptable 写了 iOS 14 小组件版的课程表，发现可用性还是不高，遂准备使用日程。

结果展示

所需环境

Python 3.7+

测试环境：Python 3.8.5, macOS 10.15.7。

步骤

感谢 SunsetYe66/ClasstableToIcal 项目。

1. 克隆项目并进入目录

git clone https://github.com/ikvarxt/ClasstableToIcal && cd ClasstableToIcal

2. 安装依赖

pip3 install uuid requests

该命令只安装了本文所需依赖，如需使用该项目其他功能，继续安装依赖 pip3 install xlrd 即可。

3. 获取课程表 json 文件

执行 getClassData.py ，按提示输入账号密码。

python3 getClassData.py

该命令如果执行成功会直接覆盖 conf_classInfo.json 文件，打开会看到自己的课程表数据。检查一下是否写入成功，再执行下一句命令。

如果该命令报错，建议再试几次。通常会成功。

注意：晚上 11 点后大佬的 API 服务器好像会关机（难道部署在了宿舍🤔），大家还是要在白天使用本脚本。

4. 生成 ics 日程文件

执行 ical_generator.py，需要输入本学期第一周周一的日期，格式：20200831，对了这就是 2020 年秋季的第一周周一日期。

python3 ical_generator.py

5. 导入日程软件

不出意外的话会生成一个命名格式类似 res-20201116T143604Z.ics 的文件，这就是最终的日程文件了，随后自行导入你在用的日程应用即可。

PS：在 iOS 设备上直接打开该日程文件会默认导入到你的 iCloud 默认日历上，我的解决方案是使用 Mac 打开，会提示你想要添加在哪个日程分组，然后 iCloud 会同步日程到手机。

End
