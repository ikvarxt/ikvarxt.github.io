好久以前就想买一个显示器，最近终于剁手了明基家的专业显示器 PD2500Q （我也是后来才知道这是一个专业的显示器，穷学生发出了没钱的惨叫），连接好电脑发现整个显示器的显示都很虚，字体也很小，降分辨率后更虚了，遂开始折腾。

本篇文章整理自知乎用户 Wenson Smith 的回答 。在此记录我的折腾过程。

利用软件 SwitchResX 开启显示器的 HiDPI 渲染模式

关闭笔记本的 SIP 系统完整性保护

进入 macOS 恢复模式

将您的 Mac 开机，然后立即在键盘上按住 Command (⌘)-R 或其他 macOS 恢复功能组合键之一。继续按住这些键，直到您看到 Apple 标志或旋转的地球。

来自苹果官网

打开 终端 ，输入 csrutil disable 禁用 SIP 。reboot 重启电脑。

安装 SwitchResX

官网下载地址 ，按提示安装，需选择 为电脑上所有用户安装 。

创建一个新的管理员用户

注销当前用户，登录新的用户。从 系统偏好设置 面板打开 SwitchResX 。

更改分辨率设置

打开你的显示器设置面板，选择 Custon Resolutions ，添加一条 Scaled resolutions 规则，设置为 2160 X 3840 (1080p HiDPI) (我是竖屏使用) 。

关闭窗口自动保存，提示输入用户密码。

重启电脑使设置生效

再次打开 SwitchResX 设置窗口，更改显示器分辨率为 1080 X 1920 (HiDPI) 即可。

奇迹出现。

使用没有问题后， 再次进入恢复模式关闭 SIP 即可 。

一些记录

  • SwitchResX 可以试用 10 天。需要购买。
  • 折腾期间电脑死机 2 次，有时会出现显示器无现实的情况，不知原因。
  • 有时设置了 Scaled Resolution 规则后，不会在列表里出现 1080 X 1920 (HiDPI) 。重启啥的，多试几次。
  • 无法任性的切换横竖显示了，会使分辨率设置失效，有时伴随电脑死机，显示器黑屏。
